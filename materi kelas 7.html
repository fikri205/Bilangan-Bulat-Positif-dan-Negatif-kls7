Materi Bilangan Bulat Positif dan Negatif

1. Definisi

Bilangan Bulat Positif: Bilangan bulat yang lebih besar dari nol. Contoh: 1, 2, 3, 4, 5, ...
Bilangan Bulat Negatif: Bilangan bulat yang kurang dari nol. Contoh: -1, -2, -3, -4, -5, ...
Bilangan Bulat Nol: Bilangan yang sama dengan nol (0).

2. Representasi Bilangan

Bilangan bulat positif terletak di sebelah kanan nol pada garis bilangan, sedangkan bilangan bulat negatif terletak di sebelah kiri nol.

3. Operasi Dasar

Penjumlahan

  • Bilangan bulat positif + Bilangan bulat positif = Bilangan bulat positif.
  • Bilangan bulat negatif + Bilangan bulat negatif = Bilangan bulat negatif.
  • Bilangan bulat positif + Bilangan bulat negatif = Hasil tergantung nilai absolut.

Contoh:

7 + (-4) = 3
-5 + (-3) = -8

Pengurangan

  • Bilangan bulat positif - Bilangan bulat positif = Bisa positif atau negatif.
  • Bilangan bulat negatif - Bilangan bulat positif = Bilangan bulat negatif.

Contoh:

-3 - 2 = -5

Perkalian

  • Bilangan bulat positif × Bilangan bulat positif = Bilangan bulat positif.
  • Bilangan bulat negatif × Bilangan bulat negatif = Bilangan bulat positif.
  • Bilangan bulat positif × Bilangan bulat negatif = Bilangan bulat negatif.

Contoh:

4 × (-2) = -8

Pembagian

  • Bilangan bulat positif ÷ Bilangan bulat positif = Bilangan bulat positif.
  • Bilangan bulat negatif ÷ Bilangan bulat negatif = Bilangan bulat positif.
  • Bilangan bulat positif ÷ Bilangan bulat negatif = Bilangan bulat negatif.

Contoh:

-12 ÷ -4 = 3

4. Aplikasi dalam Kehidupan Sehari-hari

Bilangan bulat positif sering digunakan untuk menyatakan keuntungan, suhu di atas nol, atau jumlah barang. Sementara itu, bilangan bulat negatif digunakan untuk menyatakan kerugian, suhu di bawah nol, atau utang.

  • 2. -7 + 3 = ?
  • 3. 15 - 20 = ?
  • 4. -2 × 5 = ?
  • 5. -12 ÷ -4 = ?
